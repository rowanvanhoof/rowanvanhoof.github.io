Java Markdown Conversion

As a final project in an OOP and polymorphism class, I created a rough prototype of a program that walks a filetree from a markdown processor, Obsidian, and converts each markdown file to html. The program creates a webpage for each directory that it visits and links to that directory's contents. Once this project is finished it will allow me (or anyone using obsidian notes) to share an obsidian archive with anyone, packaged as a website that is almost universally portable. Using Obsidian notes for school and computing work, I could easily share anything from an entire knowledge base of a certain topic to a portfolio of all of my code snippets and projects in a specific language.
